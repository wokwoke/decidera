# Purpose

This chapter provides a high-level overview of Decidera.

It introduces the major parts of the framework, explains how they relate to one another, and presents the overall lifecycle of a Decidera project.

Unlike later chapters, this chapter focuses on understanding the system as a whole rather than describing implementation details.

---

# Scope

This chapter covers:

- The Decidera ecosystem.
- The major system components.
- The project lifecycle.

These topics provide the conceptual map needed before exploring the detailed domain and architectural models.
