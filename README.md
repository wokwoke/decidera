# Decidera

> From discussion to decision. From decision to artifact.

## Overview

Decidera is an open-source framework for transforming structured discussions into executable artifacts.

Rather than focusing solely on code generation, Decidera emphasizes architecture, decision-making, documentation, and traceability throughout the software development process.

Its core philosophy is:

```text
Discussion
    ↓
Decision
    ↓
Artifact
```

Every discussion can produce findings.
Every finding can lead to decisions.
Every decision can generate artifacts ready for execution.

## Project Status

**Current Stage:** Design Phase

This repository currently focuses on architecture, design, and specifications before implementation begins.

## Relationship with OTONOM

Decidera is developed independently from OTONOM.

OTONOM is intended to become one of the first systems built using Decidera, while Decidera itself remains a general-purpose open-source framework.
